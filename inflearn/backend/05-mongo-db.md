### 뭐지?
- 인지도 1위 NoSQL(Not Only SQL)
- 테이블을 준비할 필요가 없다.
- JSON형태로 저장한다.
- 문서지향형 DB

### 용어
- 객체 : ducument
- collection : document들의 집합
- db : collection들의 집합

### 모델링
- 관계지향형 모델링
  - Comment Table
  - Post Table
  - Tag Table
- 문서지향형 모델링
  - Post { Comments, Tags }
